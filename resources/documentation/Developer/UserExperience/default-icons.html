3.6. Default icons

GNUstep default installation comes with a set of stock icons which can be used by the GNUstep frameworks itself or the applications you develop. There are two icons set which are delinated by their size 32*32 and 64*64.

Icons set with 32*32 size :

  • Customize (toolbar)

  • Fonts (toolbar)

  • Color (toolbar)

  • Print (toolbar)

  • Space (toolbar)

  • Flexible (toolbar)

  • Undo

  • Redo

Icons set with 64*64 size :

  • Image file

  • Sound file

  • Terminal application (not sure)

  • Web application (to be used also as a generic web icon)
