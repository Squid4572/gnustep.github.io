Position - published: 27 Jan 2009

The GNUstep Gps Navigator - by Enrico Sersale

Install
make install
The first time you run Position you must choose a default map for your location from the preferences.
Other maps can be downloaded choosing "Tools->Download map" from the main menu.
Your GPS receiver must know the NMEA protocol.
