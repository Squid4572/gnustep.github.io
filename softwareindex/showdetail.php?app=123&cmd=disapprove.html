A simple code editor for GNUstepers. - by German A. Arias

This is a simple code editor with highlight syntaxis, auto indentation and auto complete. Gemas depends on HighlighterKit framework of Saso Kiselkov. Features:

* Highlight syntax for languages C, C++, Obj-C, Obj-C++, GSmarkup.
* Highlight syntax to edit Strings, Plist, GNUmakefile, ChangeLog files and Plain text.
* Auto indentation.
* Auto complete.
* Double click above a parentheses to select all code inside these.
* Plugin to use in ProjectCenter.
* Go to Line Panel.
* Preferences Panel to change colors, editor's size, theme for the higlighter syntax...
* Menu options "App" and "Tool" to create a simple App or Tool.
* Basic indentation with TAB.
