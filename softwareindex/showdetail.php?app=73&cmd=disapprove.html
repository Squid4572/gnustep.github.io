GDL2 maps Objective-C objects to RDBMS - Port EOF Applications based on WebObjects 4.5 - by Mirko Viviani, Gerald Siebke, Manuel Guesdon, David Ayers, Matt Rice

GDL2 maps Objective-C objects to rows of relational database management systems (RDBMS). It aims to be compatible with Enterprise Objects Framework (EOF) as released with WebObjects 4.5
