PikoPixel - updates available published: 20 Apr 2018

Pixel-art editor - by Josh Freeman

PikoPixel is a free application for drawing & editing pixel-art images.
  • Easy to use
  • Unlimited undo
  • Supports multiple layers
  • Customizable canvas background
  • Hotkey-activated popup panels
  • Export upscaled images
  • Supports linear (gamma-correct) color blending

New in version 1.0 BETA9b:
  • GNUstep: Fixed window issues on Budgie, Cinnamon, GNOME, & Pantheon desktops
  • GNUstep: Various UI fixes

PikoPixel 1.0 BETA9b requires a GNUstep development environment with either of the supported compiler/runtime setups (GCC/gobjc or clang/objc2), and the following GS library versions (or later):
GNUstep Base: v1.24.9
GNUstep GUI & Back: v0.25.0
