Portability
Cocoa
GNUstep implements a significant portion of the Cocoa APIs.  
It does not currently implement some of the Core* set of  frameworks: CoreAudio, CoreData, etc.  It does, however, implement some of the CoreGraphics framework.
NIB compatibility - Cocoa .nib files from 10.2 and later are readable by GNUstep... so, when you port, you donÕt have to change anything.
pbxbuild - A tool which can buiild .xcode projects from the command line.    This tool doesnÕt require integration with Xcode SDKs since it runs on the target platform.
